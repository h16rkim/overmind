# typescript-language-server

TypeScript/JavaScript language support for Claude Code using typescript-language-server.

## Installation

### macOS (Homebrew)

```bash
./install.sh
```

Or manually:

```bash
brew install typescript-language-server
```

### Verify Installation

```bash
typescript-language-server --version
```

## Supported File Extensions

- `.ts` - TypeScript
- `.tsx` - TypeScript React
- `.js` - JavaScript
- `.jsx` - JavaScript React

## Requirements

- Node.js (for typescript-language-server to work properly)
- TypeScript (will use project's TypeScript or bundled version)
